# Account deletion in Word Battle

Effective date: March 31, 2026

## In the app

1. Open the **Account** tab.
2. Under **Account Actions**, tap **Delete Account**.
3. Read the confirmation dialog and tap **Delete Account** to confirm.

This action is permanent. You will be signed out and cannot sign in with the same credentials again, but you may create a **new** account later using the same email or Apple ID.

## What happens to your data

| Data | What we do |
|------|------------|
| Email, password, Apple sign-in | Removed from authentication (hard delete) |
| Display name | Set to **Deleted User** |
| Active tournaments | You are forfeited (same as tapping Forfeit) |
| Draft tournaments you created | Cancelled |
| Draft tournaments you joined | You leave (removed as participant) |
| Word submission text and chat messages | Replaced with `[deleted]` |
| Scores, tournament participation, tournaments you created (after cancel/forfeit) | Kept for other players, shown under **Deleted User** |

We do not merge old tournament history into a new account if you sign up again.
